### Installation
* `npm install`

### Serve
* `npm start` 

> go to [http://0.0.0.0:3000](http://0.0.0.0:3000) or [http://localhost:3000](http://localhost:3000) in your browser

### Run end-to-end tests
    
* `npm run e2e` this will start a test server and launch Protractor
